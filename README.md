# dottokei

ミニマルなドット文字で仕立てた、シンプルながら遊び心のある配信用時計素材です。

<img width="540" alt="dottokei-booth" src="https://github.com/user-attachments/assets/ce7263ae-37e0-4006-9840-b04551575158" />

## ビルド手順

プロジェクトのツールチェインとして Vite+ を使用しているので、まずはそちらをインストールしてください。
https://viteplus.dev/guide/

あとは以下のコマンドを実行するだけです。

```sh
$ pnpm install
$ pnpm run build
```
